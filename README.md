# Angular-ToDo
    <h3>This is a simple todo Angular js list app</h3>
    <h4>It can perform these features;</h4>
    <b>
    <ol>
    <li>Add todo </li>
    <li>checkbox which helps to complets todo</li>
    <li>Edit todo before finishing todo</li>
    <li>Delete todo</li>
    <li>Hide ToDo</li>
    </ol>
   <h3> Languages used </h3>
   <ul>
   <li>Html</li>
   <li>Css</li>
   <li>Javascript</li>
   <li>Angular JS</li>
   </ul>
